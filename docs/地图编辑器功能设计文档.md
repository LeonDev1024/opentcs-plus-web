## 1. 产品概述

### 1.1 产品定位

OpenTCS Web 地图编辑器是基于 OpenTCS（开放运输控制系统）的 AGV 调度管理系统核心组件，提供基于 Web 的可视化地图编辑能力，支持点位、路径、位置区域等元素的创建、编辑和管理。

### 1.2 核心价值

- **可视化编辑**：通过图形化界面创建和管理 AGV 运行地图
- **实时预览**：编辑结果即时渲染，支持多种视图模式
- **数据互通**：支持与 OpenTCS 核心系统双向数据同步
- **用户友好**：符合工业软件设计规范，操作直观高效

## 3. 已实现功能清单

### Phase 1: 交互体验优化 ✅

| 序号 | 功能 | 说明 |
|------|------|------|
| 1.1 | 批量选择功能 | Shift+点击多选、框选追加模式 |
| 1.2 | 复制粘贴 | Ctrl+C/V 复制粘贴，Ctrl+D 快速复制 |
| 1.3 | 批量删除 | Delete/Backspace 删除选中元素 |
| 1.4 | 批量属性修改 | 选中多个元素后批量修改图层属性 |
| 1.5-1.6 | 上下文菜单优化 | 完善右键菜单项（编辑/复制/粘贴/删除）|
| 1.7 | 快捷键补全 | Ctrl+A 全选, Ctrl+D 快速复制, Escape 取消选择 |
| 1.8 | 状态栏增强 | 显示选中数量、缩放比例、坐标 |
| 1.9 | 工具栏提示 | 工具按钮显示快捷键 (1-7) |

### Phase 2: 可视化增强 ✅

| 序号 | 功能 | 说明 |
|------|------|------|
| 2.1 | 点位类型图标显示 | Halt=蓝色, Park=绿色+P, Station=橙色+S, Charge=红色+⚡ |

| 2.3 | 路径动态动画效果 | 选中路径后可点击"仿真"按钮，AGV沿路径移动动画 |
| 2.4 | 位置区域图标显示 | 根据位置类型显示不同颜色和图标 |
| 2.5 | 选中状态阴影效果 | 选中元素添加红色发光阴影效果 |
| 2.6 | 鹰眼图/小地图导航 | 右下角小地图显示当前视口，支持点击跳转 |

### Phase 3: 仿真验证 ✅
